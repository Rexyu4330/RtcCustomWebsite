# RTC custom website

Un petit site web qui prend la position des bus du RTC et les affiche sur une map

## Prerequisites

Nodejs

```
axios
body-parser
express
fs
papaparse
```


## Based on

[RTC Nomade](https://m.rtcquebec.ca/) - app by [Cortex](https://cortexstudio.com/fr/agence-digitale/application-mobile-hybride)
